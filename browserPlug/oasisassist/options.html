Oasis辅助工具 - 设置

配置您的Oasis服务器地址

API 接口地址，用于插件与服务器通信（例如：http://localhost:3000/api）
Oasis 网页访问地址，留空则使用 API 地址（例如：http://localhost:3000:1249/web 或 http://localhost:3000:1249）

使用说明

  • API 接口地址（必填）：用于插件调用接口，必须配置
  • Web 网页地址（可选）：点击"打开 Oasis"按钮时跳转的地址
  • 如果 Web 地址留空，则"打开 Oasis"按钮会使用 API 地址
  • 配置示例：
    • 情况1：API 和 Web 地址相同
      → API 地址：http://localhost:3000
      → Web 地址：留空
    • 情况2：API 和 Web 地址不同
      → API 地址：http://localhost:3000/api
      → Web 地址：http://localhost:3000/web（或 http://localhost:3000）
  • 修改后需要重新打开添加页面才能生效
