Oasis辅助工具 - 设置

配置您的Oasis服务器地址

请输入完整的服务器地址（包括端口号），例如：http://localhost:3000

使用说明

  • 本地开发环境：http://localhost:3000
  • 局域网访问：http://192.168.x.x:3000
  • 生产环境：https://your-domain.com
  • 修改后需要重新打开添加页面才能生效
    • 在当前 Oasis 管理页面按 `F12` 打开开发者工具
    • 切换到 Network（网络）标签页
    • 刷新页面或点击任意菜单，查看接口请求
    • 找到任意以 `/api/` 开头的接口
    • 复制接口 URL 的前缀部分作为 API 地址
    • 接口地址：http://localhost:3000/api/navCategory/lists 插件配置：http://localhost:3000/api
